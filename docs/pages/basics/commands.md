# Commands

## `build`

Generates HTML files to the `dist` directory.

```
malta build
```

## `preview`

Runs a preview server on localhost (port 3000) for the generated site.

```
malta preview
```

```
malta preview --port 5000
```

### Options

- `--port` (`-p`): Localhost port (number - `3000` by default)
